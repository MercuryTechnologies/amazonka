data Channel Source #

A collection of data from an MQTT topic. Channels archive the raw, unprocessed messages before publishing the data to a pipeline.

See: newChannel smart constructor.

Constructors

Channel' 

Fields

  • lastMessageArrivalTime :: Maybe POSIX

    The last time when a new message arrived in the channel.

    IoT Analytics updates this value at most once per minute for one channel. Hence, the lastMessageArrivalTime value is an approximation.

    This feature only applies to messages that arrived in the data store after October 23, 2020.

    The status of the channel.

    The name of the channel.

  • retentionPeriod :: Maybe RetentionPeriod

    How long, in days, message data is kept for the channel.

  • storage :: Maybe ChannelStorage

    Where channel data is stored. You can choose one of serviceManagedS3 or customerManagedS3 storage. If not specified, the default is serviceManagedS3. You can't change this storage option after the channel is created.
